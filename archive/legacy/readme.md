\f0\fs24 \cf0 # EBYS\
\
EBYS (Eat Bugs You Spider) is a minimalist web-based electronic music radio and remix system.\
\
It functions as an automated audio processing and broadcast pipeline where uploaded tracks are:\
- separated into stems\
- analyzed for audio features\
- sliced into reusable segments\
- recombined into generative radio streams\
\
The system is designed as a terminal-like interface with minimal UI and an AI assistant ("Cricket") that guides users.\
\
---\
\
## Core Features\
\
- Audio upload system\
- Automatic stem separation (Demucs)\
- Audio analysis pipeline\
- Splice extraction engine\
- Compatibility-based remix assembly\
- Continuous web radio generation\
- Terminal-style web interface\
- AI assistant (Cricket)\
\
---\
\
## System Philosophy\
\
EBYS is a generative audio ecosystem:\
- music is treated as modular material\
- tracks are decomposed and recombined\
- the radio is continuously evolving\
- users contribute content into a shared audio system\
\
---\
\
## Tech Stack (current)\
\
- Python (processing pipeline)\
- ffmpeg (conversion)\
- Demucs (stem separation)\
- watchdog (file monitoring)\
- Max/MSP (future integration possible)\
- Web UI (terminal-style frontend)\
